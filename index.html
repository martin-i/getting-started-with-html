Concepts:
> The WEB is a collection of HTML documents.
> HTML stands for Hypertext Markup Language. It is the main type of documents in the web and these documents basically build up the WEB.
> URLs stands for Uniform Resourse Identifier and it's how different objects are refered to in the WEB.
> HTTP is the protocol that the WEB runs on.

HTML:
An HTML document is built up with elements.
Elements are constituted by an open tag, the content and a closing tag.
Tags define how the content is displayed, i.e. as a link, as an image, as text, in the same or different line, in bold or regular typography, etc.
